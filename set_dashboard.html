🏆 Composite Score — The Primary Decision Metric
0–10 scale · drives all buy, sell & rotation decisions

Every stock gets a single Composite Score (0–10) that blends technical momentum with fundamental quality. This score is the primary metric — it determines which stocks get bought first, which get rotated out, and how strong the portfolio is at any given time.

Formula
Tech Norm = (Tech Score + 3) ÷ 6 × 10 → maps −3…+3 to 0–10
Fund Score = Raw fundamental pts ÷ 13 × 10 → 0–10
Composite = Tech Norm × 60% + Fund Score × 40%
Tech +2, Fund 9.2 (AMATA)
→ (8.33×0.6)+(9.2×0.4) = 8.68/10
Tech +2, Fund 6.2 (JASIF)
→ (8.33×0.6)+(6.2×0.4) = 6.84/10
Tech 0, Fund 8.5 (GFPT)
→ (5.0×0.6)+(8.5×0.4) = 6.40/10 — HOLD
Comp ScoreAction ZoneMeaning
≥ 8.5STRONG BUYTop-tier — buy immediately, highest rotation priority
7.5 – 8.4BUYStrong — buy when slot available, rotation-in candidate
7.2 – 7.4WATCHBorderline — held positions kept, but may be rotated if better stock emerges
6.5 – 7.1WEAKBelow floor (7.2) — qualifies to rotate out; still held if no better option
≤ 6.4SELL / AVOIDWeak — not bought; held positions at high rotation risk
📅 Ex-Dividend Date Guard
If a holding's ex-dividend date is within 14 days, the system delays any SELL or ROTATION until after the record date — protecting dividend income. Stop-loss (−8%) overrides this guard.
⚡ Technical Score (−3 to +3)
gate for buy & sell signals

Three indicators each vote +1 or −1. The total (−3 to +3) acts as the gate — it must cross a threshold before a trade fires. Within the gate, composite score determines priority.

① RSI 14-day — Momentum
RSI < 30 → Oversold → +1  ·  RSI > 70 → Overbought → −1  ·  30–70 → 0
② SMA 50-day — Trend
Price > SMA → Uptrend → +1  ·  Price < SMA → Downtrend → −1
③ MACD 12/26/9 — Momentum shift
MACD > Signal → Building → +1  ·  MACD < Signal → Fading → −1
ScoreLabelAction
+3STRONG BUYAll 3 agree bullish — buy gate open
+2BUY2 of 3 bullish — buy gate open
+1WATCHWeak positive — no trade, monitor only
0HOLDMixed — no action
−1SELL2 of 3 bearish — sell gate open
−2 / −3STRONG SELLStrong bearish — sell gate open
📊 Fundamental Score (0–10)
40% weight in composite

Four financial metrics scored on graduated tiers — not just pass/fail. A stock can score high fundamentally even with a neutral tech signal.

P/E Ratio · max 3 pts
≤8 → 3 · 8–12 → 2 · 12–15 → 1 · >15 → 0
P/BV Ratio · max 3 pts
<1.0 → 3 (below book) · 1.0–1.5 → 2 · 1.5–3 → 1 · >3 → 0
ROE · max 3 pts
≥20% → 3 · 12–20% → 2 · 8–12% → 1 · <8% → 0
Dividend Yield · max 4 pts
≥9% → 4.0 · ≥8% → 3.5 · ≥6.5% → 3.0 · ≥5% → 2.5 · ≥3% → 2.0 · any div → 1.0 · none → 0
🟢 BUY Criteria
Gate conditions (all must pass)
✅ Tech score crosses to ≥ +2 (was ≤ 0 last scan)
✅ Stock not already held
✅ Portfolio has < 10 positions
Fundamental quality gate
✅ P/E < 15  ·  P/BV < 3  ·  ROE ≥ 8%
✅ Paid dividend in last 3 years
Priority — ranked by Composite Score
When multiple stocks qualify, highest composite score buys first. Two stocks both at tech +2 → better fundamentals wins the slot.
ℹ️ Only fires when score crosses from neutral (≤0) into buy territory (≥+2) — prevents re-buying the same position repeatedly within one session.
🔴 SELL Criteria
Signal sell — tech reversal
⚠️ Tech score drops to ≤ −1 (was ≥ +1 last scan)
⚠️ RSI overbought + SMA breakdown + MACD fade all confirm exit
Stop-loss — price-based, automatic
🛑 Price falls ≥ 8% below average cost
🛑 Fires immediately regardless of score or composite
Rotation — composite score weakness
🔄 Composite score below 7.2 AND a better stock (composite gain ≥ +0.8) is waiting — see Rotation section below
ℹ️ Stop-loss example: bought at ฿100 → auto-sell at ฿92 regardless of any score.
🔄 Rotation Criteria
portfolio full (10/10) · composite score drives selection

When the portfolio is full, the system checks every scan whether any holding should be swapped for a meaningfully better stock. A holding qualifies to rotate out via either trigger below — the composite score then determines which pair makes the best swap.

📉 Trigger 1 — Tech Score Fade
Held tech score dropped to ≤ +1
RSI/SMA/MACD momentum has reversed
Held ≥ 3 days
📊 Trigger 2 — Composite Below Floor
Held composite score < 7.2 / 10
Fires even if tech score is still +2
Catches fundamentally weak positions
Held ≥ 3 days
✅ Incoming Stock Must Clear
Tech score ≥ +2 (buy signal)
Composite gain ≥ +0.8 over held stock
Not in 5-day cooldown after prior rotation
3–4 days held: incoming must be +3 only
5+ days held: +2 or +3 accepted
🛡️ Anti-churn safeguards
Held loss >3% → protected, no rotation
Max 1 rotation per scan
Max 2 rotations per day
No rotation if buy/sell already fired
Ex-div within 14 days → skip rotation
ℹ️ Selection order: weakest holding (lowest composite) is targeted first. Strongest incoming (highest composite) is matched in. The swap only fires if the composite gain clears +0.8 — small differences never trigger a rotation.
