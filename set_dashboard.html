⚡ How Signals Are Scored
runs every 15 min during market hours

Each stock gets checked on 3 technical indicators. Each votes +1 or −1. Total score ranges from −3 to +3.

① RSI — 14 day
Measures momentum & overbought/oversold levels
RSI < 30 → Oversold → +1
RSI > 70 → Overbought → −1
RSI 30–70 → Neutral → 0
② SMA — 50 day
Compares current price to 50-day average trend
Price > SMA → Uptrend → +1
Price < SMA → Downtrend → −1
③ MACD — 12/26/9
Compares short-term vs long-term momentum
MACD > Signal line → Building → +1
MACD < Signal line → Fading → −1
ScoreLabelMeaning
+3STRONG BUYAll 3 indicators agree — strong upward signal
+2BUY2 of 3 indicators positive — buy signal fires
+1WATCHWeak positive — monitor only, no action
0HOLDMixed signals — no action
−1SELL2 of 3 indicators negative — sell signal fires
−2SELLStronger negative — sell signal fires
−3STRONG SELLAll 3 negative — sell signal fires
🏆 Fundamental Score (0–10) & Composite Score (0–10)
weights: Technical 60% · Fundamental 40%

The fundamental score converts each financial metric into graduated points — not just pass/fail. Combined with the technical score it produces a single composite score used to rank and prioritise which stocks to buy or rotate into.

P/E Ratio · max 3 pts
≤ 8  → 3 pts (very cheap)
8–12 → 2 pts (cheap)
12–15 → 1 pt (fair)
>15 or negative → 0 pts
P/BV Ratio · max 2 pts
≤ 1.5 → 2 pts (undervalued vs book)
1.5–3 → 1 pt (acceptable)
> 3  → 0 pts (overvalued)
ROE · max 3 pts
≥ 20% → 3 pts (excellent)
12–20% → 2 pts (good)
8–12% → 1 pt (acceptable)
< 8%  → 0 pts
Dividend · max 2 pts
Yield ≥ 3% → 2 pts (consistent & meaningful)
Paid some → 1 pt
No dividend → 0 pts
Data missing → 1 pt (benefit of doubt)
Composite Score formula
Tech normalised = (Tech score + 3) ÷ 6 × 10   → 0–10
Composite = Tech normalised × 60% + Fund score × 40%
Example A: Tech +2, Fund 8.0
→ (8.33×0.6) + (8.0×0.4) = 8.2/10
Example B: Tech +2, Fund 4.0
→ (8.33×0.6) + (4.0×0.4) = 6.6/10
Both get BUY signal, but A is bought first ↑
🟢 BUY Criteria
Technical gate (all must pass)
✅ Score jumps to ≥ +2 (was ≤ 0 last scan)
✅ Stock not already held
✅ Portfolio has < 10 positions
Fundamental gate (all must pass to buy)
✅ P/E ratio < 15
✅ P/BV ratio < 3
✅ ROE ≥ 8%
✅ Paid dividend in last 3 years
When multiple stocks qualify — ranked by Composite Score
Higher composite score → bought first. Equal technical score → better fundamentals wins.
ℹ️ Signal change rule: only fires when score crosses from neutral (≤0) into buy territory (≥+2) — prevents re-buying the same stock repeatedly on the same session.
🔴 SELL Criteria
Signal sell (score reversal)
⚠️ Score drops to ≤ −1 (was ≥ +1 last scan)
⚠️ Stock must be currently held
Stop-loss (automatic, any scan)
🛑 Price falls ≥ 8% below average buy price
🛑 Fires immediately regardless of score
ℹ️ Stop-loss example: bought at ฿100 → auto-sell if price hits ฿92 or below, no matter what the technical score says.
🔄 Rotation Criteria
runs only when portfolio is full (10/10) and no regular trade fired

When the portfolio is full, the model checks whether any current holding should be swapped for a better opportunity. Two-tier hold time rule applies.

⚡ Fast-track (3+ days)
Held stock score dropped to ≤ +1
Held ≥ 3 days
Loss not worse than −3%
New stock score = +3 only (Strong Buy)
📅 Standard (5+ days)
Held stock score dropped to ≤ +1
Held ≥ 5 days
Loss not worse than −3%
New stock score = +2 or +3
🛡️ Anti-churn safeguards
Max 1 rotation per scan (15 min)
Max 2 rotations per day
Rotated-out stock locked out for 5 days
No rotation if another trade already fired
ℹ️ What rotation does NOT do: will not sell a stock that is down >3% (locks in loss), will not rotate out a stock still scoring +2/+3 (still strong), will not fire on the same scan as a regular buy or sell signal.
